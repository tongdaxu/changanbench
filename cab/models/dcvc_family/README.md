# DCVC-family Vendored Sources

This directory contains source-only copies of DCVC video codecs used by
`cab.codec.dcvc_family`.

Large local artifacts are intentionally not included:

- pretrained checkpoints
- encoded bitstreams
- decoded frames/videos
- cache, logs, build outputs

Configure checkpoint paths in YAML, for example with `${DCVC_CKPT_DIR}`.
